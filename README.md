# OpenGanttPortal 简介
这是一个帮助让人们以一个更方便跟快捷的方式生成甘特图的项目，减少人们在手动制作甘特图花费的时间，以便达到更高的效率。
***
# 甘特图的特点：
1. 可视化程度高
2. 简单易懂
3. 强调时间管理
4. 便于进度追踪
5. 促进团队协作
6. 支持资源管理
***
# 游戏用法：
1. 下载：下载文件到本地。
2. 运行：打开文件夹，运行gobangframe即可开始游戏。
3. 开发工具：eclipse
